Game instructions:
1.The game starts with Team A batting and Team B bowling.
2.Click the "Bat" button to simulate a ball being played.
3.Runs are scored randomly between 0 and 6.
4.A score of 0 counts as a wicket (the player is out).
5.Each team plays 1 over (6 balls) by default (you can change this in script.js).
6.After Team A finishes batting, Team B will bat while Team A bowls.
7.The team with the higher total score wins.
8.Click "Reset Game" to start a new match.
Technologies used:
Html,Css,Java script
